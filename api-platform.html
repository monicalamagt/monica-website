Dense Captioning Toolkit

Overview

Scalable Python toolkit for training and evaluating dense image captioning models for vision-language models, built as part of the LLM Research group at UPenn. Integrates multimodal datasets, annotation infrastructure, and LLM-based caption generation for scientific, cultural, and 3D visuals.

What I Built

  • Scalable data pipelines for ingesting and preprocessing multimodal datasets across scientific, cultural, and 3D visual domains
  • LLM-based caption generation pipeline producing high-fidelity dense descriptions for complex images
  • Model training and evaluation framework supporting multiple VLM architectures
  • Annotation UI and backend for human-in-the-loop labeling of scientific and 3D visuals
  • Vector database integration with Qdrant for efficient retrieval across caption embeddings
